dará

O que é?

 é um jogo de estratégia para dois jogadores ou duas equipes e, alternadamente, cada jogador posiciona uma peça de cada vez no tabuleiro até completar o total das 24 peças

Como jogar?

TEM COMO OBJETIVO CAPTURAR PEÇAS ADVERSÁRIAS POR MEIO DE MOVIMENTAÇÕES ESTRATÉGICAS DAS PEÇAS PARA FORMAREM UM ALINHAMENTO TRIPLO. OU SEJA, O JOGADOR QUE CONSEGUIR POSICIONAR TRÊS PEÇAS EM UMA LINHA, VERTICAL OU HORIZONTAL, ELIMINA UMA PEÇA DO ADVERSÁRIO.
